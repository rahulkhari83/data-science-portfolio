# Hotel Operations Analytics (SQL)

Portfolio project aligned with hotel night audit responsibilities:
- revenue checks
- cancellation/no-show analysis
- ADR (average daily rate)
- payment reconciliation

**Data:** synthetic CSVs in `data/`.

## Run (PostgreSQL example)
1. Create DB (e.g., `hotel_analytics`)
2. Run schema in `sql/hotel_analytics.sql`
3. Import CSVs with `\copy`
4. Run the queries
